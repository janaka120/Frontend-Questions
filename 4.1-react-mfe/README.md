### 4.1 Micro-Frontend Router Strategy

Run Parent MFE

`cd shell`

Install dependencies

`npm install`

Start the dev server

`npm run dev`

Local server url: `http://localhost:5000/`

Run Child MFE

`cd app-a`

Install dependencies

`npm install`

Start the dev server

```
npm run build
npm run preview
```

Local server url: `http://localhost:5001/`
